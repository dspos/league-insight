常见问题

解答你的疑惑

软件通过读取英雄联盟客户端（LCU）的本地 API 工作，不修改游戏内存，不注入进程。数据来源于 Riot 官方提供的本地接口，原理上安全。但请合理使用自动化功能，避免频繁操作。

目前仅支持 Windows 10/11 (64位) 系统。由于使用了 JNA 调用 Windows API 读取进程信息，暂不支持 macOS 和 Linux。

请确保英雄联盟客户端正在运行。如果客户端已启动但仍无法连接，尝试以管理员身份运行软件，或重启客户端后再试。

理论上支持所有使用 Riot 客户端的服务器，包括国服、韩服、美服、欧服等。不支持腾讯服（如部分网吧特供版）。

在软件的「标签配置」页面，可以添加、编辑、删除标签规则。规则使用条件树结构，支持按队列、英雄、战绩数据等条件进行筛选和统计。
